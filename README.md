1. Difference between HTTP and HTTPS

HTTP and HTTPS are protocols used for communication between a browser (client) and a server.

HTTP (HyperText Transfer Protocol) sends data in plain text, meaning anyone who intercepts the data can read it. It is not secure.
HTTPS (HyperText Transfer Protocol Secure) uses encryption (SSL/TLS) to protect data. This makes the communication secure and unreadable to attackers.
2. Structure of HTTP Request and Response
HTTP Request:
Method (GET, POST, etc.)
URL/path
Headers (browser info, format, etc.)
